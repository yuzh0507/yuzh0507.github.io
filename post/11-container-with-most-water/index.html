11. Container with Most Water

2022-05-11 2 min read # C++ # LeetCode

Problem

You are given an integer array height of length n. There are n vertical lines drawn such that the two endpoints of the ith line are (i, 0) and (i, height[i]).
Find two lines that together with the x-axis form a container, such that the container contains the most water.
Return the maximum amount of water a container can store.
Notice that you may not slant the container..

Constraints

n == height.length
2 <= n <= 105
0 <= height[i] <= 104

Link: https://leetcode.com/problems/container-with-most-water/

Description
Use two iterator "left" and "right", calculate the area. If *left < *right, left++; If *left > *right, right--. record the maximum of area.

Code

class Solution {
public:
    int maxArea(vector<int>& height)
    {
        int curArea = 0;
        int maxValue = 0;

        vector<int>::iterator left = height.begin();
        vector<int>::iterator right = height.end() - 1;
        
        for (; left != right; )
        {
            if (*left == min(*left, *right))
            {
                curArea = *left * distance(left, right);
                left++;
            }
            else
            {
                curArea = *right * distance(left, right);
                right--;
            }

            maxValue = max(maxValue, curArea);
            continue;
        }
        return maxValue;
    }
};
