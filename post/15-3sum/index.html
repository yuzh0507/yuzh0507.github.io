15. 3Sum

2022-05-12 2 min read # C++ # LeetCode

Problem

Given an integer array nums, return all the triplets [nums[i], nums[j], nums[k]] such that i != j, i != k, and j != k, and nums[i] + nums[j] + nums[k] == 0.
Notice that the solution set must not contain duplicate triplets.

Constraints

0 <= nums.length <= 3000
-105 <= nums[i] <= 105

Link: https://leetcode.com/problems/3sum/

Description

  1. sort the vector.
  2. Use sortedNums[i] as the anchor, j start from i+1 to right, k start from the end of sortedNums to left.
    2.1. If sortedNums[i]+sortedNums[j]+sortedNums[k] < 0, j++
    2.2. If sortedNums[i]+sortedNums[j]+sortedNums[k] > 0, k--
    2.3. If sortedNums[i]+sortedNums[j]+sortedNums[k] = 0, record these number, and both j++ and k--;
  3. for i, j, k, after increse or decrease, once sortedNums[i] = sortedNums[i-1], keep i++.

Code

class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums)
    {
        if (nums.size() < 3)
            return {};

        vector<int> sortedNums = nums;
        sort(sortedNums.begin(),sortedNums.end());

        vector<vector<int>> sum3;
        for (int i = 0; i < sortedNums.size()-2; i++)
        {
            if (i>0 && sortedNums[i-1] == sortedNums[i])
                continue;

            int j = i+1;
            int k = sortedNums.size()-1;
            while(j < k)
            {
                if(sortedNums[i] + sortedNums[j] + sortedNums[k] < 0)
                    j++;
                else if(sortedNums[i] + sortedNums[j] + sortedNums[k] > 0)
                    k--;
                else
                {
                    sum3.push_back({sortedNums[i], sortedNums[j], sortedNums[k]});
                    j++;
                    while(j < k && sortedNums[j-1] == sortedNums[j])
                        j++;
                    k--;
                    while(j < k && sortedNums[k+1] == sortedNums[k])
                        k--;
                }
            }
        }
        return sum3;
    }
};
