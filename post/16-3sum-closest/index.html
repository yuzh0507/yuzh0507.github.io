16. 3Sum Closest

2022-05-13 3 min read # C++ # LeetCode

Problem

Given an integer array nums of length n and an integer target, find three integers in nums such that the sum is closest to target.
Return the sum of the three integers.
You may assume that each input would have exactly one solution.

Constraints

3 <= nums.length <= 1000
-1000 <= nums[i] <= 1000
-104 <= target <= 104

Link: https://leetcode.com/problems/3sum-closest/

Description

  1. sort the nums
  2. Initialize the closest (closest = accumulate(sortedNums.begin(), sortedNums.begin()+3,0); ) and difference (difference = abs(target - closest); )
  3. Use sortedNums[i] as the anchor, j start from i+1 to right, k start from the end of sortedNums to left.
    3.1. Calculate (difference = abs(target - (sortedNums[i] + sortedNums[j] + sortedNums[k])); ), refresh the closest.
    3.2. If closest = 0, return closest
    3.3. If sortedNums[i]+sortedNums[j]+sortedNums[k] < target, j++
    3.4. If sortedNums[i]+sortedNums[j]+sortedNums[k] > target, k--
    If sortedNums[i]+sortedNums[j]+sortedNums[k] = 0, record these number, and both j++ and k--;
  4. After a FOR cycling, return closest.

Code

class Solution {
public:
    int threeSumClosest(vector<int>& nums, int target)
    {
        if (nums.size() == 3)
            return nums[0]+nums[1]+nums[2];

        vector<int> sortedNums = nums;
        sort(sortedNums.begin(),sortedNums.end());

        int closest = sortedNums[0]+sortedNums[1]+sortedNums[2];
        int difference = abs(target - closest);
        for (int i = 0; i < sortedNums.size()-2; i++)
        {
            int j = i+1;
            int k = sortedNums.size()-1;
            while(j < k)
            {
                difference = abs(target - (sortedNums[i] + sortedNums[j] + sortedNums[k]));
                if (difference < abs(target - closest))
                    closest = sortedNums[i] + sortedNums[j] + sortedNums[k];

                if (closest == target)
                    return closest;            

                if(sortedNums[i] + sortedNums[j] + sortedNums[k] < target)
                    j++;
                else if(sortedNums[i] + sortedNums[j] + sortedNums[k] > target)
                    k--;
            }
        }
        return closest;
    }
};
