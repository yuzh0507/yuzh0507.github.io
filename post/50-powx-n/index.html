50. Pow(x, n)

2022-06-05 2 min read # C++ # LeetCode

Problem
Implement pow(x, n), which calculates x raised to the power n (i.e., xn).

Example

Input: x = 2.00000, n = 10
Output: 1024.00000

Input: x = 2.10000, n = 3
Output: 9.26100

Input: x = 2.00000, n = -2
Output: 0.25000
Explanation: 2-2 = 1/22 = 1/4 = 0.25

Constraints

  • -100.0 < x < 100.0
  • -231 <= n <= 231-1
  • -104 <= xn <= 104

Link: https://leetcode.com/problems/powx-n/

Description
while (n%2 != 1), x = x2, n = n/2

  • Time Complexity: O(log n)
  • Space Complexity: O(1)

Code

class Solution {
public:
    double myPow(double x, int n)
    {
        if(x == 1 || n == 0)
            return 1;

        double op = 1;

        // Transfer the value of n from negative to positive
        if (n < 0)
        {
            x = 1/x;
            // avoid -INT_MIN causes overflow
            if (n == INT_MIN)
            {
                x *= x;
                n = n/2;
            }
            n *= -1;
        }

        while (n>0)
        {
            while (!(n%2))
            {
                x *= x;
                n /= 2; 
            }
            if (n%2)
            {
                op *= x;
                n--;
            }
        }

        return op;
    }
};
