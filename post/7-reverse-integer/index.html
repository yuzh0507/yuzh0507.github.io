7. Reverse Integer

2022-05-09 2 min read # C++ # LeetCode

Problem

Given a signed 32-bit integer x, return x with its digits reversed. If reversing x causes the value to go outside the signed 32-bit integer range [-231, 231 - 1], then return 0.

Constraints

-231 <= x <= 231 - 1

Link: https://leetcode.com/problems/reverse-integer/

Description
use % to calculate the remainder, *10 to get the reverse number step by step.

Code

class Solution {
public:
    int reverse(int x)
    {
        int r = 0;
        while(x != x%10)
        {
            r = r*10 + x%10;
            x = x/10;
        }

        //check if r will exceed the boundary of int, i.e. [-2^31, 2^31-1] 
        if(r > (pow(2,31)-1)/10 || r < -pow(2,31)/10)
            return false;
        else
            r = r*10 + x%10;

        return r;
    }
};
