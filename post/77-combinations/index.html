77. Combinations

2022-06-23 2 min read # C++ # LeetCode

Problem
Given two integers n and k, return all possible combinations of k numbers out of the range [1, n].

You may return the answer in any order.

Example

Input: n = 4, k = 2
Output:
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]

Input: n = 1, k = 1
Output: [[1]]

Constraints

  • 1 <= n <= 20
  • 1 <= k <= n

Link: https://leetcode.com/problems/combinations/

Description
See code notes

  • Time Complexity: ?
  • Space Complexity: O(n*k)

Code

class Solution {
public:
    vector<vector<int>> combine(int n, int k)
    {
        vector<vector<int>> op;
        int i = 0;
        vector<int> com(k, 0);

        while (i >= 0) {
            com[i]++;

            // If the last element of com is bigger than n, back to the element before the last
            if (com[i] > n)
                i--;
            // If i reach the last of com, everytime com[i]++, push it back to op
            else if (i == k - 1)
                op.push_back(com);
            // If i haven't reach the last of com, go to the next element 
            else {
                ++i;
                com[i] = com[i - 1];
            }
        }
        return op;
    }
};
