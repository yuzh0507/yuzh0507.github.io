49. Group Anagrams

2022-06-04 2 min read # C++ # LeetCode

Problem
Given an array of strings strs, group the anagrams together. You can return the answer in any order.

An Anagram is a word or phrase formed by rearranging the letters of a different word or phrase, typically using all the original letters exactly once.

Example

Input: strs = ["eat","tea","tan","ate","nat","bat"]
Output: [["bat"],["nat","tan"],["ate","eat","tea"]]

Input: strs = [""]
Output: [[""]]

Input: strs = ["a"]
Output: [["a"]]

Constraints

  • 1 <= strs.length <= 104
  • 0 <= strs[i].length <= 100
  • strs[i] consists of lowercase English letters.

Link: https://leetcode.com/problems/group-anagrams/

Description

  1. Traverse every element of strs, sort it as the key of unordered mep
  2. Push str back to the value of unordered map with same key
  3. Push the values of unordered map back to the output

Code

class Solution {
public:
    vector<vector<string>> groupAnagrams(vector<string>& strs)
    {
        vector<vector<string>> op;
        unordered_map<string, vector<string>> anagram;

        for(auto str : strs)
        {
            string tmp = str;
            sort(tmp.begin(),tmp.end());
            anagram[tmp].push_back(str);
        }

        for(auto it = anagram.begin(); it != anagram.end(); it++)
        {
            op.push_back(it -> second);
        }

        return op;
    }
};
