6. Zigzag Conversion

2022-05-08 2 min read # C++ # LeetCode

Problem

The string "PAYPALISHIRING" is written in a zigzag pattern on a given number of rows like this: (you may want to display this pattern in a fixed font for better legibility)

P   A   H   N
A P L S I I G
Y   I   R

And then read line by line: "PAHNAPLSIIGYIR"
Write the code that will take a string and make this conversion given a number of rows:

string convert(string s, int numRows);

Constraints

1 <= s.length <= 1000
s consists of English letters (lower-case and upper-case), ',' and '.'.
1 <= numRows <= 1000

Link: https://leetcode.com/problems/zigzag-conversion/

Description
Split the 'N' into '|' '' '|'.

For the first row an the last row, they only need to put characters on the '|'. The next charcter is (2*numRows - 2) offset from the last.

For other rows, they need to put characters on the '' in addition to '|'. the middle character is (2*(numRows-curRow)-2) offset from the last '|' character.

Code

class Solution {
public:
    string convert(string s, int numRows)
    {
        if (numRows == 1)
            return s;

        string outstring ;
        
        //Start from the first row
        for(int curRow = 0; curRow < numRows; curRow++)
        {
            //Every row will add one or two character from the s
            for(int index = curRow; index < s.size();)
            {
                outstring.push_back(s[index]);
                //If this row is not the first or the last, it will additionally add a character from the s 
                if(curRow%(numRows-1) != 0)
                {
                    // Calculate the offset of the additional character from the above added character
                    int plus = 2*(numRows-curRow)-2;
                    if (index + plus < s.size())
                        outstring.push_back(s[index + plus]);

                }
                index += 2*numRows-2;
            }
        }
        

        return outstring;
    }
};
