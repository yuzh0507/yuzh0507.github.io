43. Multiply Strings

2022-05-30 2 min read # C++ # LeetCode

Problem

Given two non-negative integers num1 and num2 represented as strings, return the product of num1 and num2, also represented as a string.
Note: You must not use any built-in BigInteger library or convert the inputs to integer directly.

Example

Input: num1 = "2", num2 = "3"
Output: "6"

Input: num1 = "123", num2 = "456"
Output: "56088"

Constraints

1 <= num1.length, num2.length <= 200
num1 and num2 consist of digits only.
Both num1 and num2 do not contain any leading zero, except the number 0 itself.

Link: https://leetcode.com/problems/trapping-rain-water/

Description

  1. Calculate the product of num1[i] and num2[j]
  2. The carry is stored into sum[i+j]
  3. The unit is stored into sum[i+j+1]
  4. Ignore frontal 0, return sum

Code

class Solution {
public:
    string multiply(string num1, string num2) {
        string sum(num1.size()+num2.size(),'0');

        for (int i = num1.size() - 1; i >= 0; i--)
        {
            for (int j = num2.size() - 1; j >= 0; j--)
            {
                int tmp = (num1[i]-'0') * (num2[j]-'0') + (sum[i+j+1] - '0');
                sum[i+j+1] = tmp % 10 + '0';
                sum[i+j] += tmp/10;
            }
        }

        for (int i = 0; i < sum.size(); i++)
        {
            if(sum[i] != '0')
                return sum.substr(i);
        }

        return "0";
    }
};
